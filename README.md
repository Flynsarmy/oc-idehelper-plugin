# IDE Helpers

This plugin adds [barryvdh/ide-helpers](https://github.com/barryvdh/laravel-ide-helper) package to October for better IDE support.

## Installation

* `git clone` into */plugins/flynsarmy/idehelper*
* `composer install`
* `php artisan ide-helper:generate`
